## 📱 What is PAYO?

**PAYO** is an enterprise-grade Android Device Owner application that provides complete device management and security for loan-backed devices. It enables organizations to:

- 🔒 **Secure Devices** - Full device control with Device Owner privileges
- 💰 **Manage Payments** - Track payments and enforce payment policies
- 🛡️ **Detect Tampering** - Real-time security monitoring and tamper detection
- 📡 **Remote Management** - Control devices remotely via API
- 🔄 **Offline Support** - Works without internet with local sync
- 🚫 **Factory Reset Protection** - Prevent unauthorized device resets
- 📊 **Fleet Monitoring** - Monitor entire device fleet with heartbeat system

---

## 🎯 Key Features at a Glance

| Feature | Description |
|---------|-------------|
| **Device Owner Mode** | Elevated Android privileges for complete device control |
| **Payment Enforcement** | Soft lock reminders and hard lock for overdue payments |
| **Tamper Detection** | Detects SIM changes, boot mode, ADB, and other tampering |
| **Remote Commands** | Lock, unlock, wipe, and configure devices remotely |
| **Heartbeat System** | 30-second heartbeat for real-time device monitoring |
| **Offline Sync** | Queue operations when offline, sync when online |
| **Auto-Update** | Automatic app updates from GitHub releases |
| **Encrypted Storage** | Secure local database with encryption |
