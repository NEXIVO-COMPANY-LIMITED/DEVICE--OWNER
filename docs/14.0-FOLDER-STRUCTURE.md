## Security Enforcement Detail

### enforcement/ subpackages

| Package | Purpose |
|---------|---------|
| **adb/** | USB/debug policy (allows ADB and MTP) |
| **bootloader/** | Bootloader lock and monitoring |
| **integrity/** | Custom ROM, root, Magisk/Xposed detection |
| **input/** | Power/volume combos (fastboot, recovery, safe mode) |
| **kiosk/** | Lock task mode and foreground enforcement |
| **monitor/** | Orchestrates ADB, bootloader, custom ROM monitoring |
| **policy/** | DPM restrictions (factory reset, settings, screen capture) |

### monitoring/ subpackages

| Package | Purpose |
|---------|---------|
| **sim/** | SIM change detection |
| **tamper/** | Enhanced tamper detection, boot checker |
| **boot/** | Boot mode detection |
| **removal/** | Device Owner removal detection |
| **accessibility/** | Accessibility service guard |
| **violation/** | Device violation detection |

## Notes

- **Package**: All sources under `com.microspace.payo`
- **Language**: Kotlin (`.kt` files)
- **Build System**: Gradle with Kotlin DSL
- **Database**: Room (SQLite)
- **Networking**: Retrofit + OkHttp
- **Background Work**: WorkManager + Services
- **UI**: Jetpack Compose + XML layouts

For detailed behavior of each component, see the corresponding documentation files (0.0-18.0).
