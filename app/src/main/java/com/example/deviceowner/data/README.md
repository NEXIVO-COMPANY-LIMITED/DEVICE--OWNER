# Data Package

## Purpose
The data layer of the application, responsible for local persistence, remote API communication, and data models.

## Key Functionalities
- **Local (Room)**: Database definitions, entities, and DAOs for offline tracking.
- **Remote (Retrofit)**: API service definitions and client configuration.
- **Models**: Unified data structures for heartbeats, logs, and device state.
- **Repository**: Single source of truth for accessing data across the app.
